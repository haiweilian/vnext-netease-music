# vnext-netease-music

没错又是仿网易云，那么多了网易云项目了还写？纯粹是为了学习罢了。

之前学习的 `Vue3`、`Vite2`、`TypeScript` 一直没有新项目可用，控制不住自己的小手了必须写写，也为了要看源码熟悉熟悉语法。

整体代码写的比较简洁，功能也比较简洁，想练习的可以继续扩展，很多功能都没做。
